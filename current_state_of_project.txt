PROJECT GOAL
=============
Study how disease-specific biological context changes kinase-substrate
association (KSA) prediction. Incorporate liver cancer context into a
heterogeneous biological graph and evaluate whether it changes kinase
prioritisation in biologically meaningful ways.

SECTION 3. NODE2VEC + PREDICTION MODEL
========================================
Status: Completed.

Key design decisions:
  - node2vec runs ONCE on a PSP-stripped graph (LEAKAGE FIX)
    PSP 'phosphorylates' edges removed from embedding graph; kept as labels only.
    Non-PSP KSA edges from other sources kept even if they duplicate a PSP edge.
  - Training matrix, scoring cache built ONCE; each trial is LR mask + fit only.
  - ThreadPoolExecutor (threads share memory), n_jobs_outer=8.

Embedding: Node2VecStrategy (modular ABC; swap by implementing EmbeddingStrategy.fit)
  dimensions=32, walk_length=10, num_walks=25, workers=4, p=q=1 (DeepWalk mode)

Primary metric: adjusted_held_out_rank
  Other known true co-kinases for the site are REMOVED from the scored list
  before measuring the held-out kinase's rank.
  e.g. site has k1/k2/k3, k1 held out -> scored: [k7, k4, k2, k1]
       drop k2 -> [k7, k4, k1]   adjusted rank of k1 = 3 (not 4).

Progress reporting: progress_fn callback fires inside as_completed loop in
  parallel_utils.py, so elapsed/ETA are live measurements (not post-hoc).

SECTION 8. HUB BIAS FINDING -- CRITICAL
=========================================
The top-1 kinase frequency reveals a severe degree bias in all correlation networks:

  Network     | Top-1 monopoly (% of all 6,909 trials)
  ------------+--------------------------------------------------
  Generic     | ULK1 76.5%, WNK1 21.0%, ZAP70 2.5%
  Control     | VRK2 94.8%, YES1 4.8%
  Cancer      | VRK3 95.5%, VRK2 3.8%
  Liver FC    | ULK1 100.0% -- every single trial

Adding correlation edges makes specific kinases so highly connected in the
embedding graph that node2vec concentrates embedding mass on them, and they
dominate top-1 predictions for ALL sites regardless of biology.

The improved mean/median ranks reflect the true kinase moving from rank 300+ to
rank 50-100 -- not reaching rank 1. The hub kinase still wins top-1 everywhere.

This confirms the supervisor's prediction about node2vec degree sensitivity.
Fix: swap Node2VecStrategy for a degree-normalised embedding via the
EmbeddingStrategy ABC (no other code changes needed).


SECTION 9. TTK CASE STUDY
===========================
TTK (Mps1, mitotic kinase, liver-cancer relevant), 21 LOO trials:

  Network     | held_out mean rank | top-1 always
  ------------+--------------------+---------------
  Generic     | 247.7              | ULK1
  Control     | 25.7               | VRK2
  Cancer      | 112.7              | VRK3
  Liver FC    | 242.6              | ULK1

Control dramatically improves TTK rank (mean 25 vs 248 in generic) despite
VRK2 winning top-1 every time. Liver FC does NOT help TTK (243 vs 248).
Interpretation: TTK substrates are stably co-expressed in healthy liver tissue
but show divergent fold-change patterns in tumors, so FC edges do not capture
the TTK substrate co-regulation pattern.
