PROJECT GOAL
=============
Study how disease-specific biological context changes kinase-substrate association
(KSA) prediction.

Most existing KSA prediction methods are context-free (static sequence motifs,
generic interaction networks, previously known phosphorylation patterns).
Phosphorylation signaling is highly dynamic and context-dependent.

This project incorporates liver cancer context into a heterogeneous biological graph
by integrating phosphoproteomics, proteomics, pathway data, protein interactions,
coevolution data, and phosphosite fold-change correlation structure.

Main scientific question:
Does adding liver cancer context change kinase prioritization in biologically
meaningful ways?


SECTION 1. GENERIC GRAPH
=========================

Status: Completed and functional.

Node types (intentionally only two):
- protein nodes  (e.g. PROTEIN:AKT1)
- phosphosite nodes  (e.g. SITE:FOXO3-T32)

Kinases and phosphatases are proteins with metadata flags:
is_kinase, is_phosphatase, protein_role. They are NOT separate node types.

Sources: kinase-substrate edges, phosphatase-substrate edges, protein-site
ownership edges, pathway-based protein/site associations, PPI, PTMcode coevolution.

Size: ~13,318 nodes, ~1.1 million edges


SECTION 2. LIVER-SPECIFIC GRAPH
=================================

Status: Completed and functional.

Extends the generic graph by:
1. Adding liver-observed proteins (even if absent from generic graph)
2. Adding liver-observed phosphosites (even if absent from generic graph)
3. Reconnecting new nodes via pathway, PPI, PTMcode integration
4. Adding liver-specific phosphosite fold-change correlation edges

Size: ~16,605 nodes, ~2.1 million edges
(+3,287 liver-derived nodes, ~+1M context edges vs generic)


SECTION 3. LIVER CONTEXT MODELING
===================================

Status: Functional. Core scientific contribution.

Strategy: phosphosite fold-change correlation
1. Compute healthy control mean per phosphosite
2. Compute tumor fold-change vectors
3. Correlate fold-change vectors across tumor samples
4. Keep strong correlations at 80th percentile threshold
5. Add correlation edges (site_corr_fc_pos, site_corr_fc_neg)

Statistics:
- ~3,245 unique phosphosites used
- ~10.5 million pairwise correlations evaluated
- ~976,855 liver correlation edges added


SECTION 4. PROTEIN CO-EXPRESSION
==================================

Status: DISABLED. Partially implemented.

After preprocessing, only a very small number of proteins retained valid
control/sample measurements — too few edges to be useful.
Disabled pending preprocessing clarification and supervisor guidance.


SECTION 5. NODE2VEC + PREDICTION MODEL
========================================

Status: Completed and functional. Redesigned for speed (May 2026).

Model: logistic regression on pair features
(kinase embedding, site embedding, absolute difference, elementwise product)

Evaluation: leave-one-out edge-level LOO
Metrics: top1/5/10/20 accuracy, mean/median rank,
         held_out_rank, best_true_rank (best rank among ALL known true kinases)

Key design (post-redesign):
- Graph built ONCE per run (not per trial)
- Embeddings computed ONCE per run
- Training feature matrix built ONCE (_build_training_matrix)
- Scoring feature cache built ONCE for all unique test sites (_build_scoring_cache)
- Each trial: numpy mask (microseconds) + LR fit + predict_proba only
- ThreadPoolExecutor in parallel_utils.py (threads share memory, no pickling)

Result of redesign: 50-trial run dropped from ~24 hours to ~2-3 minutes.
6,909-trial run dropped from ~36 hours to ~5 minutes total (both models).

SECTION 8. KEY SCIENTIFIC FINDING
===================================

The two evaluations together tell a coherent and scientifically defensible story:

GENERAL trials (random sample, mostly sparse/novel sites):
  Liver graph wins clearly (mean rank 118 vs 157).
  These include less-studied, potentially liver-specific phosphosites where
  the generic graph is sparse. Liver correlation edges fill the gap.

MULTI-KINASE trials (well-studied hub phosphosites):
  Generic graph wins clearly (best_true median rank 39 vs 63 for liver).
  Hub sites like AKT1-S473, GSK3B-S9, TP53-S15 are already richly covered
  by PPI, pathway, and coevolution databases. The ~1M liver correlation edges
  reflect tissue co-regulation patterns, not direct kinase-substrate biology,
  which dilutes the signal for these well-connected hubs.

Interpretation:
  Liver-specific context adds the most value exactly where the generic graph
  is sparse. The generic graph retains advantage where prior database knowledge
  is dense. This is the expected behavior of a context-specific graph and is
  itself the scientific result.

  The generic model beats random at top-10 (12.9% vs 9.1%) and top-20
  (29.3% vs 17.5%). The liver model beats random only at top-20 (22.8% vs
  17.5%), suggesting correlation edges do not help for hub sites.

This finding drives the abstract.

SECTION 11. PROJECT SUMMARY
=============================

Fully operational framework for:
- Heterogeneous phosphorylation graph construction (generic and liver-specific)
- Disease-specific signaling context via fold-change correlation
- node2vec embeddings + logistic regression kinase-site ranking
- Large-scale LOO evaluation (6,909 trials in ~5 minutes after redesign)
- Generic vs context-aware comparison

Primary scientific contribution:
Liver-specific context changes kinase prioritization in a site-type-dependent
way — improving predictions for sparse/novel sites while the generic graph
retains advantage for well-studied hub phosphosites. This context-dependence
is itself a biologically meaningful and novel result.
